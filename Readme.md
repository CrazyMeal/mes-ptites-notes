# Mes p'tites notes🧐

Ce repository me sert à stocker diverses notes prises avec l'outil [Obsidian](https://obsidian.md/).
Il n'a pas spécialement de vocation à être partagé, mais il n'y a rien de bien particulier à cacher non plus, le repository est donc en public 😉

Dedans il y a de tout, des notes de lectures, des reminders sur des sujets comme le streaming etc... bref, tout ce qui m'interesse de prêt ou de loin (je suis assez curieux donc ça peut faire le grand écart 😅).

# Contributing
Il s'agit de notes "personnelles, mais s'il est ouvert, c'est que les contributions par exemple pour y apporter des corrections, des compléments ou autre seront bien sûr prises en compte !
Pas de guideline particulière si ce n'est de rester respectueux dans les échanges / pull-request / issues.
